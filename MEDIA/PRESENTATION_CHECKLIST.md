# FLX1.0 Presentation Checklist

Use this checklist before sharing the private evidence repo with a reviewer.

## Include In A Review

- `README.md` at the repository root.
- `VALIDATION_CARD.md`.
- `VALIDATION_RESULTS.md`.
- `BENCHMARKS.md`.
- `MEDIA/architecture_overview.png`.
- `MEDIA/cards/validation_result_card.png`.
- `MEDIA/experiment_evidence_wall.png`.
- `MEDIA/VIDEO_STORYBOARD.md`.
- `evidence/acceptance_20260606_full/acceptance_report.md`.
- `evidence/acceptance_20260606_full/acceptance_summary.json`.
- `evidence/acceptance_20260606_full/manifest.json`.
- `MEDIA/MEDIA_MANIFEST.json` for presentation-media hashes.

## Explain Clearly

- The media is presentation support.
- The evidence archive contains the actual run artefacts.
- The plots are generated from acceptance outputs.
- The validation card is a summary of the recorded acceptance status.
- The media manifest hashes the presentation files.
