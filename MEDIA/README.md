# Media

This folder is reserved for public-facing media.

Recommended files:

- architecture overview image;
- validation result screenshot;
- plot screenshots from public-safe reports;
- short demo video link;
- public presentation thumbnail.

Suggested video flow:

1. FLX1.0 runs a deterministic hardware workload.
2. A CPU-reference run is produced for the same workload.
3. The results are compared.
4. Restore/recovery is demonstrated.
5. The final report shows zero mismatches for the validated case.
